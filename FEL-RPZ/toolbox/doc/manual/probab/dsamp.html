DSAMP

Generates samples from discrete distribution.


 Synopsis:
  X = dsamp(Px,num_data)

 Input:
  Px [dim x 1] Discrete probability distribution; it must
   satisfy sum(Px) = 1 and min(Px) >= 0.
  num_data [1 x 1] Number of samples to be generated.
 
 Example:
  Px = [0.2 0.3 0.1 0.4];
  X = dsamp(Px,1000);
  Px_emp = hist(X,4)/1000

 See also
  GSAMP, GMMSAMP
