SIGMOID

Evaluates sigmoid function.


 Synopsis:
  y = sigmoid(X,model)
  y = sigmois(X,A1,A2)

 Description:
  y = sigmoid(X,model) returns 
     y = 1/(1+exp(A(1)*X + A(2))

  where A = model.A.

  y = sigmois(X, A1, A2) allows A to be given as A = [A1 A2].

 Input:
  X [1 x num_data] Inputs.
  model.A [2 x 1] Sigmoid parameters.

 Output:
  y [1 x num_data] Evaluated sigmoid function.

 See also
  MLSIGMOID.
