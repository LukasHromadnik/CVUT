TRAIN_OCR

Training of OCR classifier based on multiclass SVM.


 Description:
  The following steps are performed:
   - Training set is created from data in directory ExamplesDir.
   - Multi-class SVM is trained.
   - The trained SVM model is saved.
    
Source: train_ocr.m
