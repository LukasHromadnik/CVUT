Image denoising using kernel PCA. 

 make_noisy_data    - Adds Gaussian noise to USPS database.   
 show_denois_tuning - Plots curves of tuning stage of Kernel PCA.
 show_denoising     - Image denosing of USPS hand-written numerals.
 train_kpca_denois  - Training of kernel PCA model for image denoising.  
 train_lin_denois   - Training of linear PCA model for image denoising.

 USPSModelGreedyKPCA.mat - Kernel PCA model of USPS numerals.
 USPSModelLinPCA.mat     - Linear PCA model of USPS numerals.
 noisy_circle.mat        - Toy data generated by 'gencircledata'.
